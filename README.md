# WhispeRu

WhispeRu — приложение для Windows, предназначенное для офлайн-распознавания русской речи в реальном времени и голосового ввода текста. Программа принимает сигнал с микрофона, преобразует речь в текст с учетом пунктуации и вставляет результат в активное поле ввода без переключения между приложениями.

## Назначение

Приложение разработано для сокращения времени на ручной набор текста в повседневных рабочих сценариях. WhispeRu может использоваться для подготовки сообщений, документации, комментариев, заметок и другого текстового контента непосредственно в целевом окне программы.

## Основные функции

- офлайн-распознавание речи на локальном компьютере
- преобразование русской речи в текст в реальном времени
- автоматическая вставка распознанного текста в активное окно
- настраиваемая глобальная горячая клавиша для управления записью
- режим удержания клавиши и режим переключения записи
- автоматическая пунктуация и капитализация
- опциональное преобразование числительных в цифры
- типографические замены кавычек и тире
- выбор микрофона
- звуковые сигналы начала и окончания записи
- поддержка сохранения результата в буфер обмена
- история диктовок с возможностью очистки или отключения

## Постобработка текста

WhispeRu поддерживает постобработку распознанного текста с помощью правил нормализации и автозамены. После распознавания приложение может применять обычные текстовые замены и правила на основе регулярных выражений. Встроенные голосовые команды позволяют управлять пунктуацией и форматированием, включая точки, запятые, переносы строк, абзацы, скобки и кавычки.

## Типовой сценарий использования

1. Установить курсор в любое текстовое поле.
2. Нажать назначенную горячую клавишу для начала записи.
3. Продиктовать текст.
4. Остановить запись.
5. Получить автоматическую вставку распознанного текста в активное приложение.

## Целевая среда использования

- Microsoft Windows 10
- Microsoft Windows 11
- сценарии ввода текста на русском языке
- настольные приложения со стандартными текстовыми полями

## Конфиденциальность

WhispeRu ориентирован на локальную обработку данных. Распознавание речи выполняется на устройстве пользователя. Для базового использования приложения регистрация не требуется, а настройки хранятся локально на компьютере.

## Ограничения

- поддерживается только русский язык
- в отдельных приложениях автоматическая вставка может быть ограничена
- поддерживается только платформа Windows

## Планируемые расширения

- распознавание аудио- и видеофайлов
- поддержка дополнительных языков
